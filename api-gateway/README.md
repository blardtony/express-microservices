# Express API Rest

## Install

Run this command to install **node_modules**

```bash
npm insall
```

Run this command to launch this app with docker and build images
```bash
docker compose up --build
```

Or this one
```bash
docker-compose up --build
```

**--build** This option build images before executing docker compose
